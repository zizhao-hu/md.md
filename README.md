# md.md

The entire project is one file: [`md.md`](md.md).

A static instruction md fits nobody — research, projects, and jobs each drift on their own schedule. Rather than scaffold a skill that tries to guess what you need, this file is a one-line seed. The model reads it, hits a mismatch between what the file says and what the work actually requires, and rewrites the file. The only invariant: whatever shape it takes, it must still point back to itself as the seed so the next iteration can edit it too.

No plugin, no hook, no skill. Just a file that authors itself.
